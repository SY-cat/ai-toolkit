# AI Toolkit 🛠️

> AI 驱动的开发者工具集 — 由 GitHub Copilot CLI 与 WorkBuddy Agent 协作构建

## 🎯 项目简介

AI Toolkit 是一个纯前端开发者工具集合，提供常用的编码、解码、格式化、哈希等工具。本项目完全由 AI Agent（WorkBuddy + GitHub Copilot CLI）协作生成，作为 AI 辅助开发能力的实际演示。

## ✨ 功能特性

| 工具 | 说明 |
|------|------|
| 📋 JSON 格式化 | 压缩/格式化 JSON 数据 |
| 🔐 Base64 编解码 | 文本与 Base64 互转 |
| 🔑 SHA-256 哈希 | 计算文本哈希值 |
| ⏱️ 时间戳转换 | Unix 时间戳与日期互转 |
| 🎯 正则表达式测试 | 实时匹配正则表达式 |
| 🎨 颜色格式转换 | HEX ↔ RGB ↔ HSL 互转 |

## 🚀 使用方式

### 本地运行
```bash
# 方式一：直接打开
open index.html

# 方式二：使用 serve
npx serve .
```

### 在线访问
部署到 GitHub Pages 或 Vercel 即可获得在线演示地址。
